==============================================================================
  PART 3: PERIODIC BOUNDARIES — WHERE EXPONENTIAL WOULD WORK
==============================================================================

  With periodic boundary conditions (Eisenstein torus):

  • Every node has exactly 6 neighbors
  • Each node has neighbors in all 3 chirality classes (0, +1, -1)
  • Non-detection rate ≈ p_int ≈ 0.5% for ALL nodes
  • No boundary leakage channel

  In this case, the Peierls argument applies cleanly:
  • An undetectable pattern must simultaneously evade detection
    at ALL incident edges × ALL time steps
  • With p_det ≈ 0.995 per single error, the probability of a
    weight-w pattern being entirely undetected is ≤ (0.005)^w
  • The number of connected weight-w patterns ≤ n · μ^w
  • So ε_L ≤ n · Σ_{w≥d} (μ · ε · 0.005)^w

  For this to converge: ε < 1/(μ · 0.005) ≈ 43
  i.e., the threshold would be essentially unlimited (always below it)

  With periodic boundaries, d would grow with linear dimension L,
  giving genuine exponential suppression:
    ε_L ≤ n · (μ · ε · 0.005)^L ∝ exp(-c · L)

  HOWEVER: The paper uses OPEN boundaries (finite lattice cells).
  Physical realization likely requires finite cells, so the open
  boundary analysis is the honest one.
